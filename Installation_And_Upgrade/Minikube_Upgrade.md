# Delete running Minikube instance  
1. Open a shell.  
```
minikube delete  
```
2. Delete the minikube folder.  
3. Use the windows installer - works fine.  You can find it on the page in the below link.  
https://kubernetes.io/docs/tasks/tools/install-minikube/
